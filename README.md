# Лабораторная работа №2: Сравнение моделей регрессии для прогнозирования цен на жилье

## Описание работы
В лабораторной работе проводится сравнительный анализ пяти моделей регрессии для прогнозирования цен на жилье. Исследуется влияние масштабирования признаков, отбора признаков и регуляризации на качество предсказаний.

## Датасет
- Файл: house_price_regression_dataset.csv
- Размер: 1000 записей, 8 признаков
- Целевая переменная: House_Price (цена дома)

## Проведенные эксперименты

1. **Ridge регрессия (α=4) + StandardScaler** - с L2-регуляризацией
2. **Линейная регрессия (2 признака)** - только Square_Footage и Lot_Size
3. **LinearRegression (без масштабирования)** - базовая модель
4. **LinearRegression (удалены слабые признаки)** - удалены Num_Bathrooms, Neighborhood_Quality, Num_Bedrooms
5. **LinearRegression + StandardScaler** - с масштабированием признаков

## Результаты

| Эксперимент | MSE | RMSE | MAE | R² |
|-------------|-----|------|-----|-----|
| Ridge регрессия (α=4) + StandardScaler | 42364618272 | 205826 | 171138 | 0.5626 |
| LinearRegression + StandardScaler | 42364618272 | 205826 | 171138 | 0.5626 |
| LinearRegression (без масштабирования) | 42364618272 | 205826 | 171138 | 0.5626 |
| LinearRegression (удалены слабые признаки) | 42406025385 | 205927 | 171032 | 0.5621 |
| Линейная регрессия (2 признака) | 46325683990 | 215210 | 178640 | 0.5217 |

## Анализ результатов

- Лучший результат показали модели с полным набором признаков (RMSE ≈ 205,826)
- Удаление трех признаков незначительно ухудшило результат
- Модель только с двумя признаками показала худший результат (R² = 0.52)
- Масштабирование признаков не повлияло на качество линейной регрессии
- Ridge с регуляризацией не улучшил результат по сравнению с обычной линейной регрессией

## Выводы

1. Оптимальная модель - линейная регрессия со всеми признаками
2. Модель объясняет около 56% дисперсии цен (R² = 0.56)
3. Средняя ошибка предсказания составляет ≈ $171,000
4. Наиболее важные факторы: площадь дома и размер участка
