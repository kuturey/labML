# Лабораторная работа №2: Сравнение моделей регрессии для прогнозирования цен на жилье

## Описание работы
В лабораторной работе проводится сравнительный анализ пяти моделей регрессии для прогнозирования цен на жилье. Исследуется влияние масштабирования признаков, отбора признаков и регуляризации на качество предсказаний.

## Датасет
- **Файл**: house_price_regression_dataset.csv
- **Размер**: 1000 записей, 8 признаков
- **Целевая переменная**: House_Price (цена дома)
- **Признаки**:
  - Square_Footage (площадь в кв. футах)
  - Num_Bedrooms (количество спален)
  - Num_Bathrooms (количество ванных комнат)
  - Year_Built (год постройки)
  - Lot_Size (размер участка)
  - Garage_Size (размер гаража)
  - Neighborhood_Quality (качество района)

## Проведенные эксперименты

1. **Ridge регрессия (α=4) + StandardScaler** - с L2-регуляризацией
2. **LinearRegression (без масштабирования)** - базовая модель
3. **LinearRegression + StandardScaler** - с масштабированием признаков
4. **LinearRegression (удалены слабые признаки)** - удалены Num_Bathrooms, Neighborhood_Quality, Num_Bedrooms
5. **Линейная регрессия (2 признака)** - только Square_Footage и Lot_Size

## Результаты экспериментов

| Эксперимент | MSE | RMSE | MAE | R² |
|-------------|-----|------|-----|-----|
| Ridge регрессия (α=4) + StandardScaler | 106,351,510 | 10,313 | 8,086 | 0.9982 |
| LinearRegression (без масштабирования) | 108,162,680 | 10,400 | 8,111 | 0.9982 |
| LinearRegression + StandardScaler | 108,162,680 | 10,400 | 8,111 | 0.9982 |
| LinearRegression (удалены слабые признаки) | 386,501,326 | 19,660 | 15,891 | 0.9936 |
| Линейная регрессия (2 признака) | 901,423,377 | 30,024 | 24,062 | 0.9850 |

## Анализ результатов

- **Лучший результат**: Ridge регрессия с масштабированием (RMSE = 10,313, R² = 0.9982)
- **Масштабирование не влияет** на линейную регрессию (идентичные результаты с/без StandardScaler)
- **Удаление слабых признаков** ухудшило результат (RMSE вырос до 19,660)
- **Двух признаков недостаточно** - модель показала худший результат (R² = 0.985)
- **Ridge не улучшил** результат относительно обычной линейной регрессии (значит переобучения нет)

## Выводы

1. **Лучшая модель**: Ridge регрессия с масштабированием (RMSE = 10,313)
2. **Качество предсказания**: модель объясняет 99.82% дисперсии цен (R² = 0.9982)
3. **Важность признаков**: 
   - Square_Footage - самый важный признак (корреляция 0.99)
   - Lot_Size - второй по значимости (корреляция 0.16)
   - Остальные признаки слабо влияют на цену
4. **Удаление признаков** даже с низкой корреляцией ухудшает точность
5. **Ridge-регрессия** показала результат чуть лучше обычной линейной регрессии
